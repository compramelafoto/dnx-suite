# Diseñador PRO — Atajos de teclado

- **Escape:** Deseleccionar frame.
- **Delete / Backspace:** Borrar el frame seleccionado (undo lo recupera).
- **Ctrl/Cmd + Z:** Deshacer.
- **Ctrl/Cmd + Shift + Z:** Rehacer.

Los atajos no se aplican cuando el foco está en un campo de texto (input, textarea, select o contenteditable).

*(En el canvas, Espacio + arrastrar = pan; Ctrl/Cmd + +/- = zoom, manejados dentro de CanvasEditor.)*
